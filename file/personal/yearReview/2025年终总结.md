# 概述

25年评级后感到失落<!--more-->原因如下：

1.主要是对后半生的焦虑

2.对工作中没有成绩的失落

3.公司人际关系没有维护好的失落

如何理解与解决：

1.首先明白一点不可能工作到退休，所以一定要关注自己。考编和创业才是退路

2.想要获得成绩拼的是精力和时间，你只是效率高没用，要花费更多的时间和精力投入到工作，做出超过普通同事的成绩，才有可能脱颖而出。这也并非绝对，还要看领导的态度。所以你敢去搏一下吗？我不愿意

3.在合理的范围内多聊多相处，尽可能搞好关系，特别是领导。当然也要知道，人与人之间是有眼缘的，实在融不进去也没关系，不是你的问题。

今后要怎么做：

1.考编创业：要重视和多花时间精力

2.努力保住工作：想想如何与领导多接触
