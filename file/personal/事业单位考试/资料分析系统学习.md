# 资料分析

- 掌握解题技巧
- 大量刷题







# 大纲解读

* 要求：对文字、图表的理解和分析加工的能力
* 材料形式：图表、数字、文字材料
* 考察的是对有效信息的提取能力(很重要)







# 问题与解决方案

- 读不懂题目：学习统计术语
- 找不到材料中对应内容：需要结构性阅读
- 列不对计算公式：针对重点题型学习
- 算的慢：掌握速算技巧
- 目标是正确率控制在85%以上,控制在1-1.5分钟一题,15题控制在23分钟内







# 结构阅读
- 核心解题步骤
  - 读题：划出时间、关键词
  - 列式：基于题目列出计算公式
  - 计算：从题目找数据，代入公式计算
- 文字性材料
  - 解题步骤
    - 快速阅读,找出关键词
    - (非常重要)找准问题中关键词,判断出是什么类型问题
    - (非常重要)先列公式,再找数据[复杂的数据可以用列表的形式写出来]
  - 注意事项
    - 文字陷阱与数据隐蔽性
- 表格材料
  - 先看标题
  - 在看横纵坐标
  - 注意看单位：终点注意单位与其它不同的特殊单位，有可能是坑
- 图形材料
  - 重点阅读：标题、横轴、纵轴、图例、单位（注意特殊单位）
  - 饼状图问题
    - 如何看图:默认从12点钟方向顺时针排列
    - 如何做题
      - 总体已知,看特殊角/特殊比例
      - 总体未知:看大小关系、倍数关系













# 速算技巧

- 截位直除法(材料分析的根本解法)
  - 截位:从左往右截,截几位保留几位(四舍五入),从非0开始截.例如0.023678截两位24截三位237
  - 截谁
    - 一步直除：例如只需要除一次（A/B）,只截分母
    - 多步连除：例如（a/b）/（c/d）,分子分母同时截
  - 截几位
    - 选项差距大:截两位
      - 首位均不同
      - 首位相同,最接近的两个选项次位的差值也很大(差值>首位)
    - 选项差距小:截三位
      - 首位相同,最接近的两个选项次位的差值小(差值<=首位)

- 分数比较(核心让分数的比较条件一致即分子或分母相同)
  - 一大一小直接看:分子大且分母小则大
  - 同大同小
    - 直除首位:比较13/78与27/124的大小，第一个直除首位(130/78)是0.1+，第二个直除首位(270/124)是0.2+
    - 倍数关系：比较13/78与27/151的大小，13到27变化了2倍+，78到151变化了2倍-，因此后者最大。

- 特殊分数(需要倍)

  - 与1/2的关系

    - 1/8=12.5%,1/16=6.25%
    - 1/3=33.3%,1/6=16.7%,1/12=8.3%

  - 互相转化

    - 1/7=14.3%,1/14=7.1%

    - 1/9=11.1%,1/11=9.1%

  - 等差记忆

    - 1/13=7.7%,1/15=6.7%
    - 1/17=5.9%,1/18=5.6%,1/19=5.3%

- 特殊数字

  - A * 1.1 = A+A的0.1倍
  - A * 0.9 = A - A的0.1倍
  - A * 1.5 = A + A的一半


- 分数相乘
  - 方法一：a%*b%=(a*b)/100%
  - 方法二：一个化为分数，一个不变

















# 统计术语

- 基期：作为对比参照的时期
- 现期：相对于基期的称为现期
- 增长量:从基期到现期的变化量
- 增长率:增长量/基期

- 同比：与历史同期相比较（同比与环比最主要的区分是统计周期的区分，只有环比才会有）
- 环比：现有统计周期与上一个统计周期相比较
- 统计周期：例如材料中给到2019年9月份产值为100亿，则可以得知统计周期是一个月，环比是与上个月相比

- 比重：某部分在总体中所占的百分比。部分/总体
- 增长贡献率:部分增长量/整体增长量
- 倍数：如第一产业产值是第二产业产值的几倍
- 平均数
- 百分数:增长率、比重、利润率等
- 百分点：百分数的单位。例如去年第一产业比重10%，今年第一产业比重11%，则今年比去年增加了1个百分点
- 成数：例如三成表示十分之三，七成五表示75%
- 翻番：翻一番为原来的2倍，翻n番为原来的2的n次方倍
- 降幅:比较的是绝对值,与正负号无关
- 变化幅度:即增长率的绝对值,与正负号无关
- 利润率:利润/收入
- 产销率:销量/产量
- 顺差：出口>进口，挣钱了表示顺差
- 逆差：出口<进口，花钱了表示逆差
- 国内生产总值GDP：=第一产业（农业）增加值+第二产业（工业和建筑业）增加值+第三产业（其它行业）增加值。增加值不等于增长量，增加值就理解为GDP即可
- 五年计划：十二五（11年-15年），十三五为（16年到20年），十四五（21年到25年）
- 恩格尔系数：食品支出占个人消费总支出的比重。
  - 这个系数越底表示生活水平越高
  - 大于60%为贫穷
  - 50-60%为温饱
  - 40-50%为小康
  - 30-40为相对富裕
  - 20-30为富裕
  - <20%为极其富裕
- 基尼系数：代表收入差距的指标。数据介于0-1之间，越大标识不平等程度越高。
  - 0.4为国际红线
  - <0.2绝对平均
  - 0.2-0.3比较平均
  - 0.3-0.4相对合理
  - 0.4-0.5差距大
  - 大于0.5极大

- 注意事项
  - 计算时单位换算有可能是坑
  - 17年产值比16年增加了280%=17年产值比16年增加了2.8倍=17年产值是16年的3.8倍

















# 重点题型

## 基期量与现期量

- 基期量=现期/(1+r)
- A/(1+r)可以估算为A * (1-r)<===>A/(1-r)可以估算为A * (1+r)误差可以忽略不计
- 现期量=基期 * (1+r)
- 基期量=增长量/增长率









## 增长量

* 计算公式
  * 增长量=现期-基期
  * 增长量=基期*r
  * 增长量=(现期xr)/(1+r)
  * 年均增长量=增长量/间隔年数
* 速算(非常重要)
  * 增长率百分化:即将r转换为分数形式1/n(注意一定要是分子为1的形式)
    * 增长量=现期/(n+1) <==> r为正数
    * 减少量=现期/(n-1) <==> r为负数
  * 增长率百分化之取中法
    * 遇到百分数左右为难,不好计算,选择最相近的一个特殊分数
    * 例如10.5%介于1/10和1/9之间,取1/9.5
  * 增长率百分化之公式法
    * 实在想不起来时,n=100/百分号前的数字 (保留小数点后一位)
    * 例如:42%:n=100/42=2.3,则42%=1/2.3
* 增长量比较
  * 识别:增长最多/最少,下降最多/最少
  * 区别:增长最快/最慢-->指的是增长率的比较
  * 解题方法
    * 如果有柱状图直接看高度差
    * 已知现期量与增长率
      * 同大则大(看的是r的绝对值,若r是负数就是降幅大)
      * 一大一小百分化(利用百分化技巧)
      * 一大一小,分子大、分母小















## 增长率

* 考察题型
  * 给百分点,加减法:高减低加
  * 给具体量,套公式
    * r=增长量/基期
    * r=(现期量-基期量)/基期量
    * r=增长量/(现期量-增长量)
    * r=现期/基期 - 1
    * 年均增长率:现期量=基期量*(1+年均增长率)的n次方，其中n为相差年份
* 间隔增长率
  * 识别:中间隔了一年,求增长率
  * 例如09年增长率是r1，10年的增长率是r2，问相比08年10年的增长率是多少？
  * r间隔=r1+r2+r1xr2
    * r1和r2绝对值都小于10%,则乘积小于10%,可以忽略
    * 若两者都较大,则可以化为分数求解
  * 间隔倍数=间隔增长率r+1
  * 间隔基期=现期/(1+间隔增长率r)

* 年均增长率
  * 识别:年均增长最快/最慢、年均增速排序
  * <u>(1+r)的n次方=现期量/基期量,n为年份差</u>
  * 基期的确定(n的确定)
    * 具体时间段:例如2011-2015,基期为2011,那就是4
    * 五年规划:例如十四五,基期为2020年,就是5
  * 常考题型
    * 比较大小:直接比较“现期量/基期量”即可
    * 真的需要计算(性价比太低),直接带入验证即可
* 混合增长率
  * 定义:当整体由多个部分组成且每个部分有不同的增长率,混合后即为混合增长率
  * 识别:(重点就是识别)已知两部分增长率、求整体增长量
  * 常见混合情况
    * 房产+地产=房地产
    * 电信+邮政=邮电
    * 1-2月+3月=一季度
  * 解题技巧
    * 口诀:混后居中,偏量大
    * 线段法:量与距离成反比(参考数量关系),一定要画线段(数形结合)

















## 比重

* 比重分类
  
  * 现期比重:当前部分占整体的百分比
  * 基期比重:难点是需要根据现期数据计算出基期数据
  * 两期比重:考察的是两个时期比重的变化量/变化率
  
* 现期比重

  * 与题目给定时间相同
  * 计算公式(截位直除)
    * 比重=部分/总体
    * 总体=部分/比重
    * 部分=总体*比重

* 基期比重
  
  * 问题识别:问题时间在材料之前,结合“占、比重”
  * 案例：今年第一产业产值A，增速为a，全国产值B，增速为b，求去年第一产业占全国的比重
  * 公式：<u>比重=[A/(1+a)] / [B/(1+b)]=(A/B)x[(1+b)/(1+a)] 即交叉书写</u>
  * 计算技巧：分开估算，后者只计算与1的大小即可。计算的时候直接斜着写好记忆
  
* 两期比重差
  
  * 识别:<u>两个时间</u>,一个比重,升降
  * 升降判断
    * 比较部分增长率与整体整长率,部分大则上升,否则下降(比较时带正负号)
    * 案例：全国产值增速3%，第一产业产值增速2.9%，第二产业产值增速3.2%，问第一产业与第二产业比重是上升还是下降？
    * 解法：部分增速与整体增速的大小相比较，大则增大，小则减小
  * 公式计算
    * <u>(A/B)x[(a-b)/(1+a)]</u>
    * 公式中字母均为现期数据(<u>该公式实际上是现期比重-基期比重的推导</u>)
    * 案例：今年第一产业产值A，增速为a，全国产值B，增速为b，求第一产业的比重比上一年增加或减少多少百分点
  
  * 解题步骤：
    * 1、先根据自身增速与整体增速来判断比去年增加还是减少
    * 2、由解题公式得知A/B乘一个比1小的数，进而推导出选项
    * 3、一般前两步就可以解题，如果不行需要根据公式取估算，不需要详细计算















## 平均数

利用截位直除法进行速算

核心如何快速找到A、a、B、b

- 平均数

  - 基础公式:平均数=总量A/份数B
  - 特殊案例：A*x+B*y=平均数*(x+y)例如全国居民平均收入与城镇居民收入+农村居民收入题型

- 现期平均数

  - 以当前时间数据计算出的平均数
  - <u>(重要)计算公式:后/前</u>
    - 人均收入=收入/人数
    - 单位面积产量=产量/面积
    - 平均每人次客运旅客运输距离=运输距离/人次
  - 计算技巧:结合选项差异、截位直除

- 基期平均数

  - <u>基期平均数=基期量/基期份数</u>
  - 若已知现期量、增长率
    - <u>基期平均数=(A/B)x[(1+b)/(1+a)]</u> 
    - A为现期量总数,a为A的增长率,B现期量总个数,b为B的增长率
    - 算左看右:左边A/B计算,右边只要看于1的关系,再排出选项

- 两期平均数

  - 题目特点:比较<u>两个时间段</u>平均数之间的关系,通常会问上升或下降
  - 识别
    - 题目中涉及两个时间、平均(每/单位)
    - 问升降:两个时间、平均、升降(高低)
    - 问增长率:平均/每/单位、增长了%
    - 问增长量:平均/每/单位、增长、单位

  - 升降判断
    - 比较分子增长率与分母整长率,分子大则上升,否则下降(比较时带正负号)
    - 例如:单位总收入下降3%,单位员工下降5%,问人均收入上升还是下降?-3>-5
  - 平均值增长量(两期平均数差值)
    - 平均数增长量=现期平均数-基期平均数
    - (A/B)*[(a-b)/(1+a)]
  - 平均数增长率/增速
    - 例如：房地产销售额增速为a，销售面积增速为b，求均价增速为多少
    - 基础公式:<u>平均数增长率=(现期平均数-基期平均数)/基期平均数</u>
    - 推导公式：<u>(a-b)/(1+b)</u>
    - a表示分子增长率(总数)、b表示分母整长率(份数)
    - 特别注意:两期比重算的是差值,两期平均数算的是增长率,是比值











## 倍数

- 倍数

  - A是B的n倍,则A=nB
  - A比B高n倍,则A-B=nB

  - 倍数=增长率+1=增长倍数+1
  - 倍数是A是B的多少倍,增长倍数是A比B多的是B的多少倍

- 比值

  - A与B的比值,即A:B=A/B

- 现期倍数

  - 识别:问题时间与材料时间保持一致
  - A是B的几倍:A/B
  - A比B多几倍:A/B-1 = (A-B)/B 相当于增长率
  - 整长率=倍数+1

- 基期倍数

  - 识别:时间在材料时间之前,结合倍数的问法(几倍)
  - 已知现期量和增长量
    - 基期A是基期B的几倍=(A-A的增长量)/(B-B的增长量)
    - 基期A比基期B多几倍=(A-A的增长量)/(B-B的增长量) - 1
  - 已知现期量和增长率
    - 基期A是基期B的几倍=(A/B) * (1+b)/(1+a)
    - 基期A比基期B多几倍=(A/B) * (1+b)/(1+a) - 1
  - 示例：18年第一产业产值A，增速a，第二产业产值B，增速b，求17年第一产业是第二产业产值的几倍？
  - 公式：<u>(A/B) * (1+b)/(1+a)</u>
