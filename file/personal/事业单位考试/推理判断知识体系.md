# 学习方案

- 掌握解题技巧

- 观看相关视频，整理知识点

- 大量刷题

# 图形推理题型
- 一组图
  - 顺序找规律，很少是跳跃找规律
- 两组图
  - 第一组找规律，第二组应用
- 九宫格
  - 横向找规律，第一组找，第二组验证，第三组应用
- 分组类
- 空间重构

# 学习的重点
- 如何快速定位题目的考点
- 位置规律
  - 识别方法：元素组成相同
  - 考点：平移、旋转、翻转
  - 平移
    - 有方向性：沿直线、绕圈
    - 有步数：恒定、递增
  - 旋转
    - 顺时针、逆时针、常见角度
  - 翻转
    - 左右翻转、上下翻转
- 样式规律
  - 识别方法：元素组成相似
  - 加减同异
    - 特点：线条重复出现
    - 相加、相减
    - 去同求异
    - 去异求同
    - 在进行加减同异运算后再旋转/翻转
    - 在进行旋转/翻转后在进行加减同异
  - 黑白运算
    - 特点：内部颜色不同
    - 相同位置运算
    - 黑白运算：黑与白的或与运算
    - 黑块数量相同，优先平移
    - 黑块数量不同，优先黑白运算
- 属性规律
  - 识别特征：元素组成不同、不相似
  - 对称性
    - 轴对称：方向、数量、旋转
    - 中心对称：按照对称点旋转180度与原图相同
    - 找出所有的对称轴+中心，然后解题
  - 曲直性
    - 全曲线
    - 全直线
    - 曲+直
    - 直+曲
  - 开闭性
    - 全封闭
    - 全开放
- 数量规律
  - 识别特征：元素组成不相同、不相似
  - 点数量
    - 图形特征
      - 线条交叉明显
      - 一团线交叉
      - 相切较多
    - 线与线的交点
    - 直直焦点/直曲焦点/曲曲焦点
    - 线与圆的切点
    - 圆内交点
  - 线数量
    - 数线
      - 数直线/曲线
      - 数曲线+/-直线
      - 数线的平行或者方向
    - 笔划
      - 特征图形：五角星、“日/田”及变形、圆和相切/相交
      - 一笔划成：线条联通+奇点(有奇数条线链接的点，端点也是奇点)数=0或2
      - 多笔画：笔画数=奇点数/2
    - 汉字：数笔划、某种笔划的数量
  - 面数量
    - 面的特点：封闭空间+白色区域
    - 分割区域比较明显时考虑
    - 大面与小面相似时也可以考虑
    - 图形中特殊图形的面数
  - 元素数量
    - 特征：有多个独立的小图形组成
    - 元素种类
    - 元素个数
    - 元素替换
  - 部分数量
    - 特征：生活化的图形、黑色粗线条图形
    - 数组成部分的数量
  - 角数量
    - 角指的是内角，0到180度
    - 特征：扇形图、改造图、折线图
    - 只数锐角、钝角、直角
- 特殊规律
  - 功能元素
    - 点：观察点对图形的标记作用、点与点的联系（如对称轴）
    - 箭头：观察箭头指向、箭头之间的联系
  - 图形关系
    - 相离、相压（注意上下关系）
    - 相交于面、相交于点
    - 相交于边(数量、最长边、最短边)
  - 2

# 黑白块专项梳理
- 平移：移动，内外圈移动，翻转，旋转
- 数部分：黑白各自几部分
- 对称：黑白各自对称，对称轴方向
- 角链接：出现在黑白方块图中，通过角链接
- 一笔划
- 黑白异或运算：一般出现在九宫格中


# 六面体橡皮法
- 解决立方体图形考题
- 可以买一块，带进去
- 实在不行用手指


# 空间重构
- 六面体
  - 相对面
    - 同行或同列相隔一个面就能形成相对面
    - Z字型的两端形成相对面
  - 相邻面
    - 最重要的是找上下左右
    - 直角边折叠后是同一条边，即确定相邻面
    - 折叠前后相近关系不变
    - 两个面的公共边是唯一的
  - 画边法
    - 找特殊面的唯一点或边
    - 顺时针描边
    - 描边确定相邻面，用排除法
  - 六面体橡皮法
- 四面体
  - 一条直线上的两条边折起来后是一条边
  - 箭头法：确定左右下三个面
  - 通过方向性找相邻面
- 八面体
  - 钝角形成的两条边是同一条边
  - 每一个顶点都发出四个面且只有四个面
  - 箭头法：确定方向性的相邻面
  - 公共点：确定相连的四个面


# 其它图形
- 三视图
  - 主视图，俯视图，左视图
  - 一定是平面图
- 截面图
  - 被切部分的形状
  - 六面体：可以切矩形、梯形三角形（只能锐角、钝角）
  - 圆柱体：可以切圆、椭圆、矩形
  - 圆锥体：可以切圆、椭圆、三角形
  - 圆台体：可以切圆、椭圆、梯形
- 拼合图形
  - 凹凸必有对应

# 类比推理
- 语义关系
  - 近义词
  - 反义词
  - 二级辨析：一级关系选不出唯一答案时
    - 感情色彩：褒义、贬义、中性
    - 词语结构：例如大同小异
  - 比喻意义/象征意义
- 逻辑关系
  - 全同关系：即完全相等的意思
  - 矛盾关系：两个对立面，如生、死
  - 并列关系：如黑白或者苹果香蕉
  - 种属关系：A是B的一种
  - 组成关系：A是B的组成
  - 交叉关系：例如女性与公务员
  - 关联关系：小麦与馒头（材料关系），白酒与发酵（工艺关系），银行卡与支付（功能关系），盐与咸（属性关系）
  - 因果关系：失之毫厘 差以千里
- 语法关系
  - 主谓关系：如学生和学习
  - 动宾关系：如学习和功课
  - 主宾关系：如学生和功课
  - 如何解题：尽量用少的词语造句子

# 定义判断
- 找到最佳定义的答案
- 解题思路
  - 看清楚题目问的是什么
  - 找关键词关键句
  - 选择最符合的选项
- 找主体和客体
  - 主体：行为的发出者
  - 客体：行为的接受者
- 引导性的关键句
- 包含定义的句子优先看
- 注意阅读补充说明

# 逻辑判断
- 翻译推理
  - 特点：题目有逻辑性词语表述，根据题目可以推断出什么（例如：如果……那么）
  - 解题方法
    - 先翻译，再推理
    - 谁是必要条件，谁在箭头后面
  - 前推后：前面一句话推导出后面的动作A->B（反过来不一定能推导）
    - 典型用例：只要……就……，如果……那么……，所有……都……，……是……的充分条件，……就/则/都/一定/……
  - 后推前：只有B才A(可以得出A->B)，不……不……，……才……，除非……否则不……，……是……的必要条件
  - 且和或：A且B(两者都满足)，A或B(两者至少一个满足或都满足)，要么A要么B（只能满足一个）
  - 逆否等价：A->B等价于-B->-A
  - 德摩根定律：-(A且B)=-A或-B，-(A或B)=-A且-B
- 排列组合
  - 特点：给出多组对象，回答多组对象的关系
  - 排除法：读一句，排除一个
  - 代入法：选项代入题干去验证
  - 辅助技巧
    - 以最多出现的词作为推理起点
    - 符号：大于、小于，题目中有比较关系
    - 列表格
- 日常结论
  - 特点：题目中没有逻辑词或可套用的技巧，比较平直的叙述
  - 解题：不要无中生有，不要偷换概念，有绝对词/程度词的表述
- 逻辑论证（重点）
  - 论点：作者所持的观点，对问题提出的见解、主张、看法等
  - 论据：用来证明论点的事实或依据
  - 论证：用论据证明论点的过程
  - 找论点
    - 关键词：因此、所以、由此推出……
    - 首尾句
    - 结合提问方式，问什么找什么
  - 加强类题型
    - 题型特点：如“以下哪项是上述论断的前提/假设/必要条件”“以下哪项加强/支持/赞同/证明上述论点”
    - “搭桥”法加强论证：建立论据与论点的联系
    - 补充论据
      - 补充必要条件：选项为论点成立的必要条件（没他不行）
      - 解释原因：说明论点成立的原因
      - 举例支持：证明论点成立的例子
- 真假推理
