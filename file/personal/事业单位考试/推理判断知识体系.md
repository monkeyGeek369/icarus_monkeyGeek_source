# 学习方案

- 掌握解题技巧

- 观看相关视频，整理知识点

- 大量刷题

# 图形推理题型
- 一组图
  - 顺序找规律，很少是跳跃找规律
- 两组图
  - 第一组找规律，第二组应用
- 九宫格
  - 横向找规律，第一组找，第二组验证，第三组应用
- 分组类
- 空间重构

# 学习的重点
- 如何快速定位题目的考点
- 位置规律
  - 特点：元素组成相同
  - 考点：平移、旋转、翻转
  - 平移
    - 有方向性：沿直线、绕圈
    - 有步数：恒定、递增
  - 旋转
  - 翻转
