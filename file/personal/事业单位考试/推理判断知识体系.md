# 推理判断

- 掌握解题技巧
- 观看相关视频，整理知识点
- 大量刷题











# 类比推理

- 概念理解

  - 根据一组词中属性相同推理出另一组词也具备相同属性
  - 层层辨析:先进行一级辨析再进行二级辨析
  - 大众思维,不要专牛角尖
  - 题型:两词型、三词型、对当型、相交组合

  

- 语义关系
  - 近义反义关系:注意积累成语
    - 二级辨析
      - 感情色彩：褒义、贬义、中性
      - 语意递进
      - 积极消极
      - 词语结构：例如大同小异
  - 比喻象征意义
    - 常识积累
      - 南冠-囚犯、桑梓-家乡、青衿-读书人、中山浪-忘恩负义、桃李-学生、鸿雁/筏札-书信、总角-小孩、康乃馨-母爱









- 逻辑关系(重要)
  - 全同关系：完全相等的意思
  - 并列关系：同一层级且地位相等的完全不同事物.如黑白或者苹果香蕉
    - 矛盾关系:只有两个对立面，如生、死
    - 反对关系:在一个整体中除了有ab还有cd
  - 包含关系:两个事物一大一小,可以造句区分
    - 种属关系：A是B的一种
      - 特指:造句后指向明确,如美国:国家
      - 泛指:造句后指向模糊,如葡萄:水果.并未说明葡萄是巨峰、阳光玫瑰
    - 组成关系：A是B的组成
  - 交叉关系：事物存在交集,使用“有的”造句来确定.例如女性与公务员
  - 对应关系
    - 材料关系:小麦与馒头
    - 功能关系:银行卡与支付
      - 二级辨析:主要功能、次要功能、是否唯一功能
    - 地点关系:嵩山与河南
    - 方式目的关系:过滤与净水
    - 属性关系(重点):盐与咸
    - 工艺关系:白酒与发酵
    - 因果关系：失之毫厘 差以千里
      - 必然结果、或然结果、积极结果、消极结果
    - 顺承关系:写稿与投稿
      - 二级辨析:主体是否一致









- 语法关系
  - 主谓关系：主语+谓语,如学生和学习
  - 动宾关系：谓语+宾语,如学习和功课
  - 主宾关系：如学生和功课
  - 偏正关系:形容词,例如肥沃的土壤
  - 并列关系:例如踏踏实实
  - 解题技巧
    - 尽量用少的词语造句子
    - 二级辨析:词性(名词/动词)





- 词语拆分
  - 当辨析完一级关系后且符合拆词条件则可拆分
  - 两两看关系、拆成单字看关系
  - 拆词条件
    - 相同的单子重复出现:“温:温暖:温室”
    - 成语拆分:“争分:夺秒”,还可再继续拆,例如分:秒、争:夺
    - 词和词之间无任何逻辑关系:成败:呼吸可以拆为成:败、呼:吸











# 图形推理
- 命题形式
  - 一组图:顺序找规律，很少是跳跃找规律
  - 两组图:第一组找规律，第二组应用
  - 九宫格:横向找规律，第一组找，第二组验证，第三组应用
  - 分组分类:123属于一个规律,456属于另一个规律
  - 空间重构:例如三视图





- 位置规律
  - 识别方法：元素组成相同
  - 平移(可以适当对移动对象进行标记序号方便观察)
    - 有方向性：沿直线、顺时针、逆时针
    - 有步数：恒定、等差(2/2/2...)、周期(2/4...)
  - 旋转
    - 顺时针、逆时针、常见角度
  - 翻转
    - 左右翻转、上下翻转











- 样式规律
  - 识别方法：元素组成相似
  - 遍历
    - 特点:相同元素重复出现
    - 形状遍历:空缺处是否缺少某个应有的形状
    - 颜色遍历:空缺处是否缺少某个应有的颜色/线条
  - 加减同异
    - 特点：线条重复出现
    - 相加、相减
    - 去同求异
    - 去异求同
    - 在进行加减同异运算后再旋转/翻转
    - 在进行旋转/翻转后在进行加减同异
  - 黑白运算
    - 特点：内部颜色不同
    - 相同位置运算
    - 黑白运算：黑与白的或与运算
    - 黑块数量相同，优先平移
    - 黑块数量不同，优先黑白运算









- 属性规律
  - 识别特征：元素组成不同、不相似
  - 对称性
    - 轴对称：方向、数量、旋转
    - 中心对称：按照对称点旋转180度与原图相同
    - 找出所有的对称轴+中心，然后解题
  - 曲直性
    - 全曲线
    - 全直线
    - 曲+直 / 直+曲
  - 开闭性
    - 全封闭 / 全开放(途中任意一点都可以跑出图形) / 半封闭半开放















- 特殊规律
  - 功能元素
    - 标记作用:标记特殊位置特殊元素
      - 标记点：曲直交点、直直交点、曲曲交点
      - 标记线:标记长短、标记曲直
      - 标记角:锐角/钝角/直角、最大角、最小角
      - 标记面:大小面、单独面、相交面
    - 每图都有两个功能点
      - 是否标记同一面
      - 标记点是否连通:点与点的连线（如对称轴）
    - 箭头：观察箭头指向、箭头之间的联系
  - 图形关系
    - 相离: 外离、内涵
    - 相交于点:曲/直、位置、数量
    - 相交于边:数量、最长边、最短边、曲/直、整体/部分
    - 相交于面:形状(最长考)、与最大图形是否相似、相压（注意上下关系）













- 数量规律
  - 识别特征：元素组成不相同、不相似
  - 点数量
    - 图形特征:线条交叉明显、一团线交叉、相切较多
    - 曲直交点:直直交点/直曲交点/曲曲交点
    - 内外交点:框内、框上、框外交点(特征图看是否都有圆)
    - 线与圆:切点、交点
  - 线数量
    - 数线
      - 数直线/曲线
      - 数曲线+/-直线
      - 内图线/外框线
      - 数线的平行或者方向
    - 笔划
      - 特征图形：五角星、“日/田”及变形、圆相切/相交
      - 一笔划成：线条联通+奇点(有奇数条线链接的点，端点也是奇点)数=0或2、所有的端点都是寄点
      - 多笔画：笔画数=奇点数/2
  - 面数量
    - 面的定义：空白封闭区域
    - 特征图:分割区域比较明显、生活化的图形、粗线条图形中留白区域
    - 常规考法:数面数
    - 细化考法
      - 所有面形状是否都相同
      - 部分面:相同面的个数、特征面(三角形/四边形/扇形)、面的数量与最大/最外图形边数的关系
      - 单个面:样式/形状、大小面相似
    - 面与面的相交特性:点、线、图形
  - 元素数量
    - 特征：有多个独立的小图形组成
    - 解法:种类、个数、位置、替换
    - 元素相同都能组合为一个图形或者立方体
    - 图一到图三相邻两图在相交/相同元素上的数量一致
  - 部分数量
    - 特征：生活化的图形、黑色粗线条图形
    - 数组成部分的数量
  - 角数量
    - 特征:三角形、锯齿状折线、有图形缺口、扇形图、改造图、折线图
    - 角定义:线与线形成的内角，0到180度
    - 角类型:锐角、钝角、直角(常考)















- 汉字类型
  - 开闭特性
  - 结构特性:上下、左右、半包围
  - 共同笔画
  - 内部面数







- 黑白块专项梳理
  - 平移：移动、折返跑、循环跑，内外圈移动，翻转，旋转
  - 数部分：黑白各自几部分
  - 对称：黑白各自对称，对称轴方向
  - 角链接：出现在黑白方块图中，通过角链接
  - 一笔划
  - 黑白异或运算：一般出现在九宫格中





- 空间重构
  - 相对面
    - 同行或同列相隔一个面
    - Z字型两端
  - 相邻面
    - 公共边法
      - 两个面的公共边是唯一的
      - 直角边折叠后是同一条边，即确定相邻面
    - 四面共线
      - 四个在一条直线上的面,首尾两条边是一条边
    - 方向确定
      - 找到有方向性的特殊面
      - 确定特殊面的上下左右面
    - 画边法
      - 找某一特殊面,确定特殊面唯一定点(这里的唯一指的是对所有面来讲都是唯一的)
      - 顺时针描边,确定相邻面顺序
      - 知道相邻面顺序后再看选项做题
  - 解题方法
    - 六面体橡皮法
    - 先标记面,再解题
    - 找相对面
    - 找相邻面
  - 截面图
    - 被切部分的形状
    - 六面体：可以切矩形、梯形三角形（只能锐角、钝角）
    - 圆柱体：可以切圆、椭圆、矩形
    - 圆锥体：可以切圆、椭圆、三角形
    - 圆台体：可以切圆、椭圆、梯形

  - 拼合图形
    - 凹凸必有对应

















# 事件排序

- 题型:给出五个相互独立事件,按照正常逻辑或最少假设排序
- 排序原则:从前往后排序
- 解题原则
  - 通读提干:当阅读量不大时
  - 首尾排序:尝试确定首位
  - 逻辑关系:找对应的关系,例如因果关系、顺承关系、充分必要条件等
  - 分析选项:用带入排除法









# 定义判断(重要)
- 找到最符合或最不符合的答案
- 基本原则
  - 定义无误原则:必须严格以来提干,不能基于自己的理解或尝试
  - 择优选择原则:关键词最多、最直接的答案
- 解题思路
  - 看提问:到底是符合还是不符合
  - 提取条件:关键词、关键句
  - 对比选项:找最合适
- 识别关键点
  - 区分主客体
    - 主体-行为的发出者、客体-行为的接受者,(主客体不符合的一定不能选)
  - 行为句式区分
    - 方式目的型:通过...的方式、达到...的目的
    - 原因结果型:由于...的原因、导致...的结果
    - 条件型:当...的条件时、在...情况下、在...之前
  - 概念定义与补充说明
    - 如果文段很长可以优先看定义部分,读懂后没必要往后读
    - 补充说明往往蕴藏着容易判断的必要条件.例如即、也就是说、括号、冒号、顿号、包括、例如
  - 多定义
    - 文中有多个定义,只要找到问题定义
- 做题小技巧
  - 拆词:找到定义中的限定词,例如公益广告可以拆出”公益“
  - 同构选项排除:选项中出现两个在意思和结构上相同的选项













# 逻辑判断
- 真假推理
  - 题型：题目中有真有假，真假不确定
  - 矛盾关系：找到相互矛盾的表述/选项
    - 常考公式
      - A和-A
      - 所有和有的:“所有是 和 有的不是”、“所有不是 和 有的是”
      - A->B 和 A且-B
      - A且B 和 -A或-B
      - A或B 和 -A且-B
    - 解题思路：一真其余假，一假其余真
  - 反对关系
    - 上反对:所有A都是B 和 所有A都不是B （不能同真,因此必定有一假，可同假）
    - 下反对:有的A是B 和 有A的不是B（不可同假,因此必有一真，可以同真）
  - 翻译案例
    - 甲和乙至多有一人参加活动:-甲 或 -乙













- 组合排列
  - 特点：给出多组对象，回答多组对象的关系
  - 排除法：读一句，排除一个
  - 代入法：选项代入题干去验证
  - 信息推理
    - 无法使用代入法、排除法时使用,找到开头顺着往下捋
    - 从确定信息往下捋
    - 从最大信息入手:以最多出现的词作为推理起点
    - 使用假设法:没有确定信息/最大信息时使用
      - 若假设与题干条件不违背则假设可能成立
      - 若假设与题干条件违背则证明起点错了,再从起点推理
  - 数形结合
    - 使用>=<等数学符号来表示年龄、成绩、收入等大小比较关系进行推理判断
    - 列表格:设计多个对象的组合排列时













- 翻译推理
  - 特点：题目有逻辑性词语表述，根据题目可以推断出什么（例如：如果……那么）
  - 解题方法
    - 看清选项,选是还是选非
    - 先翻译，再推理
    - 某些题目给出多个限制条件，需找出各个条件交叉最多的对象，先进行分析
  - 前推后
    - 只要A就B:A->B（反过来不能推导）
    - 典型用例：如果……那么……，所有……都……，……是……的充分条件，……就/则/都/一定/……
  - 后推前
    - 只有A才B:(B->A)
    - 除非A否则不B:(B->A),例如除非写作业否则不能看电视.“看电视->写作业”
    - 除非A否则B:(-B->A),例如除非交电费否则就断电.“-断电->交电费”
    - 不……不……、……才……
    - 逻辑变形
      - 必要条件、先觉条件、基础、前提、必要假设、必不可少
      - 谁是必要条件，谁在箭头后面
  - 逆否等价(极重点)
    - A->B等价于-B->-A
  - 传递关系
    - 已知A->B,B->C,C->D可以得出A->B->C->D,不要简写为A->D
  - 且和或：
    - 且的等价关系词:和、即...又...、不仅...而且...、不但...还要...、我很丑但是我很温柔
    - 关系要点
      - A且B(两者都满足)
      - A或B(两者至少一个满足,可都满足)
      - 要么A要么B（只能满足一个）
    - 德摩根定律:-(A且B)=-A或-B，-(A或B)=-A且-B
    - 推理中的或关系如何理解?
      - 例如-a或b,即两边只能有一个为真
      - 如果-a为真,则能得出-b
      - 如果-a为假,则能得出b
  - 特殊问法:下列选项中与题干的推导方式一致/不一致的
    - 先分析题干:标记AB并得出A->B
    - 再分析选项:标记选项AB的推导关系
    - 对比题干和选项的AB推导关系











- 逻辑论证（重点）
  - 论证三要素
    - 论据：用来证明论点的事实或依据
    - 论证：用论据证明论点的过程
    - 论点：作者所持的观点，对问题提出的见解、主张、看法等
  - 找论点（最重要）
    - 结合提问方式，问什么找什么
    - 关键词：因此、所以、由此推出……
    - 首尾句
    - 注意:理解论点(找准论点中关键词)至关重要,很多错误都是不够理解论点
  - 找论据
    - 关键词:由于、因为、鉴于、根据
    - 就近原则:就在论点前后
    - 常见形式:原因、数据、事例、实验、调查研究
  - 加强类题型
    - 题型特点：如“以下哪项是上述论断的前提/假设/必要条件”“以下哪项加强/支持/赞同/证明上述论点”
    - 搭桥法：论点与论据不一致时考虑(建立论据与论点的联系)
    - 补充论据:(只有论点或论点论据话题一致时)
      - 补充必要条件：选项为论点成立的必要条件（没他不行）
      - 解释说明：说明论点成立的原因
      - 举例说明：证明论点成立的例子
    - 加强力度
      - 搭桥=必要条件>解释原因>举例
  - 削弱类题型
    - 特点：最能质疑/反驳/削弱/反对上去论述的？
    - 解题方法
      - 否定论点：论点论据话题一致时(前提是找准论点)
      - 拆桥法：论点论据话题不一致时(否定论据与论点之间的必然联系)
      - 否定论据:预测、建议类时
      - 因果论证题型:论点有因果时
        - 因果倒置：把因果搞反
        - 另有他因：不赞成也不反对，而是说另外的原因也同样导致了这个结果
    - 消弱力度
      - 否论点=因果倒置>拆桥>否论据>另有它因











- 日常结论
  - 特点：题目中没有逻辑词或可套用的技巧，比较平直的叙述
  - 解题：不要无中生有，不要偷换概念，有绝对词/程度词的表述







- 集合推理（画图的方式或许更快）
  - 所有：
    - 所有1是2 可以得出 1能推出2 以及 有的1是2
    - 可以递推：所有1是2和所有2是3 可以退出 所有1是3
  - 有的：有一个，有一部分，全部
    - 有的1是2 推不出 有的1不是2（不要被生活常识干扰）
    - 有的1是2 不能进行逆否
    - 交集：有的1是2=有的2是1
