# 基本信息

- 考察能力
  - 管理角色意识：站在管理者角色，做到不偏袒，合理、全面、公正处理问题
  - 问题分析能力：能够发现问题，产生原因，处理办法
  - 计划与控制能力：利用可支配资源、有规划有顺序的推进，解决问题
  - 沟通协调能力：利用与他人的沟通实现问题发现、问题协调，问题解决等
  - 文字表达能力：准确、清晰的进行文字表达
- 分数:四道题,150分,分数分布差不多是30、35、45、40(前低后高)
- 材料类型
  - 案例型材料：讲故事，需要总结、提炼
  - 观点型材料：主体、客体、观点、理由、推论
  - 数据型材料：1.看主体+数据（增长/下降）2.看不同主体间的对比
- 评分构成:要点分(多写不扣分)；逻辑分；格式分；卷面分
- 必须要做的:练字、阅读、做题(大量反复吃透真题)、实践
- 做题原则
  - 立足题目:看清问题、理清思路、按照要求
  - 材料为王:原文摘录、每一个答案都要有依据
  - 找准角色定位




# 剖题法

- 审题
  - 确定题型：归纳、对策、应急、沟通、公文
  - 认清身份角色
  - 找出限制：主要内容、目的
  - 读懂要求：准确、全面、条理
- 析点
  - 要素标记（问题W、背景B、原因Y、影响S、对策D）<u>要建立自己的符号体系</u>
  - 词的阅读
    - 问题词:（消极/负面/否定）问题、难题、不足、损失、耗费、缺失、缺乏、缺少、落后、瓶颈、薄弱、弊端、恶化、不容乐观、令人堪忧
    - 原因词:（导致某结果的条件）原因、因为、由于、究其根源、由……所导致的、根本上、本质上、主要是、关键在于、因素、要素、缘故、缘由、因由、起因、主要归咎于、与……有关
    - 积极影响词:（表示结果的词汇）有利于、有益于、有助于、积极意义、提高了、促进了、推动了、实现了、完成了、提升了、引发了……、具有……的重要价值、重要的作用、重要性、……为……提供了保证、深远的意义、重大的意义
    - 消极影响词:（表示结果的词汇）不利于、导致、破坏了、负面、关乎、制约了、阻碍了、损害了、危害了、严重后果、损害了……的形象、破坏了……的关系、影响了……的发展
    - 对策词：（解决办法）需要、必须、应该；措施、方案、思路、方法、途径；促进、提高、加强、完善、建立、健全、倾斜、依托、搭建、借助、狠抓、着力、优化；
    - 并列关联词:(前后一样重要)既……也(又)……；又……又……；一方面……一方面……；一边……一边……；还；也；同样；除了；此外；另一方面；同时；还有；除此之外；另外；且；并且；加之
    - 递进关联词:（重点关注后者）不但、不仅、不光、甚至、更、以至、何况、况且、尤其、甚至于；不但（不仅、不光）……，而且（并且、还、也、甚至）……；尚且（况且）……，何况……。
    - 转折关联词:（重点关注后者）虽然……但是……；虽然......却；虽说……然而……；固然……但是……；尽管…… 还是……；尽管……但是……；既便.....但......；……却……；
    - 总结词：总之、因此、所以、综上所述、这
    - 观点词：权威观点（文件、讲话、法规），网友观点（要分析判断是否可用）
    - 重点词：本质上、换言之、很大程度上
  - 句的阅读
    - 首句：总领作用，含答案要点
    - 尾句：总结作用，含答案要点
    - 中心句：段落中间，承上启下，总领段落大意
    - 核心句：由核心词汇或结论性词汇引领的句子（凸显重点）
  - 图表阅读
    - 流程图:1、标题、节点、备注等各自文字表述.2、先看主支再看分支,3、关注流向
    - 统计表
      - 标题、横轴、纵轴
      - 数据:总量、数字变化、数据对比
        - 横向对比:行业/地区/主体对比(大小、高低、快慢、占比)
        - 纵向对比:同比、环比、增幅、减幅、趋势
      - 分类概括:结合横轴/纵轴、时间、阶段、主体、范围
- 加工
  - 语言加工:核心是精简
    - 去除重复、关联词、虚词、修饰词
    - 提取关键词
  - 逻辑加工
    - 前置词:高档词、权威词、高频词
    - 逻辑呈现:通过123序号有序呈现









# 观点归纳

- 基本信息:必考、250字、35分
- 解题技巧
  - 审题
    - 问数:1问、2问(有可能问题和对策要分别解答,其实是两问)
    - 对象：归纳谁
    - 要求:例如“分条列项”、“一句话概括”(不用换行,直接序号/;号)
  - 析点：找词找句
    - 直接概括
      - 每段落提取核心句、高频词、关键词(宁滥勿缺)(原词原句)
    - 综合概括
      - 去虚保实、总结共性、保留因果
      - 少用概括，除非找不到关键词
  - 加工:核心是如何分类?
    - 题干提示:例如题干问问题和对策,那就是两类
    - 材料暗示:
      - 材料有总括句,说明了分类方向
      - 材料有时间线，需要根据时间分类
      - 材料具有逻辑/递进时（常见）
    - 内容分类
      - 同义法:意思相同、语意相近
      - 同主体法
        - 问题、原因、影响、对策
        - 宏观:国家、政府
        - 中观:社会、体制内、媒体、公众
        - 微观:个人、家庭
      - 同范围法:例如要素纬度、时间维度、过程维度
        - 五大领域:政治、经济、社会、生态、文化
        - 区域:东中西、城乡、国内、国外
        - 原因:内因、外因、主观、客观
        - 影响:利弊
  - 书写:总括句+序数词+前置词+具体要点
    - 总括句：材料中高度概括的句子（字数较少时通常要有）
    - 前置词：（字数较少时不要）同义法、同范围法、同主体法
- 常见问题归纳

  - 与人才相关问题(制/伍/能/思/基)
    - 制度机制失衡:招募标准不合理、培养错位、发展受限、考核虚化、激励失效
    - 队伍结构失衡:数量不足、老龄化、职业病
    - 能力素质低下:技能滞后、经验欠缺、专业水平不足、工作效率低、工作质量不高
    - 思想态度问题:责任与服务意识淡薄、态度不端正、工作方式不当(违反程序/缺乏沟通)
    - 基础保障不足:硬件条件简陋、软件供应不足(经费/物资)、环境艰苦









# 提出对策

- 基本信息:常考、35分左右、350字左右
- 解题策略
  - 审题
    - 判断题型：对策/方法/建议/方案/解决之道/锦囊妙计
    - 要求：针对性、可行性、操作性
    - 做法对象：对策、概括+对策
    - 问数
      - 题干明确要求：并、及、与、和（表示两问，一般问题和对策分别作答，不做对应书写）
      - 提干暗示：例如“问题概括要全面，建议要可行”
    - 身份：解决问题的角度
      - 给定身份：一般是政府工作人员
      - 未明确身份：你就是政府角度（全角度），只要是想到的对策就可以写
  - 析点
    - 对策来源
      - 直接对策：政策、权威、成功经验、建议
      - 间接对策（找到后直接标注对策）
        - 找问题：找解决方案或反写
        - 找原因：直接反写
        - 找教训：规避、否定表达
      - 自创对策：知识储备、实践经验
  - 加工
    - 形式：总括句(可省略)+各个小总括句+对策展开
    - 归类方式:题目要求、文章暗示、内容归类(同内容、同范围、同主体)
  - 书写
    - 要点前置：核心得分词放前面
    - 适度扩展：主体+手段+核心词
    - 联系实际：掌握常见对策
- 常见对策
  - 加强宣传引导(创新/媒/实/活)
    - 创新式宣传：网络直播/公益广告/条幅签字
    - 媒介宣传：三微一端(微博/微信/微视/客户端)/电视/广播/报纸/短信
    - 实物宣传：展板/板报/条幅/手册
    - 活动宣传：宣讲会/座谈会/报告会/展销会/文艺演出/征文/知识竞赛/演习/"六进〞(进机关、进社区、进农村、进企业、进学校、进媒体）
  - 加大监督力度
    - 加强内部监督：完善监督机制、强化监督执纪力度、加强监督考核、推行“双随机、一公开〞 监管模式等
    - 强化社会监督：舆论监督、群众监督、拓宽举报渠道
  - 加强管理力度(明/界/评/长)
    - 奖罚分明:红黑榜制度、奖惩机制
    - 责任界定:细化分工、岗位责任制
    - 评价与改进:考核考评、问责机制、容错纠错
    - 长效保障:长效机制
  - 加强人才队伍建设(引/培/使/激/留)
    - 精准引进:优化招聘机制、拓宽引才渠道
    - 系统培育:完善培养体系、加强思想建设、以老带新提升业务能力、职业生涯规划
    - 科学使用:完善用人制度、做到人岗相适、明确岗位职责
    - 人才激励（考/待/收/发/典/匠）:制定考核与晋升机制、强化待遇保障、优化收入分配机制、拓展发展空间、选树典型、弘扬工匠精神
    - 用心留住（生/文/保）:优化服务生态、注重人文关怀、完善配套保障
  - 加大资金投入(来/使/管)
    - 资金来源:申请专项资金、争取财政拨款、引入社会资本、银行贷款、自有资金、经营性收入、众筹
    - 资金使用:固定资产投资、日常运维保障、未来发展投资
    - 管理与监督:健全管理制度、内部审计、外部监督、建立绩效评估、对外公示
  - 多部门协调联动(机/流/总/聚/沟)
    - 建立协调机制:联调联动机制、成立专家组、联席会议、明确责权、设定共同目标、制定考核标注
    - 优化工作流程:简化流程、联合办公、信息共享、推行一站式服务
    - 总结反馈:做好过程记录、事项督办、事后评估与优化
    - 聚焦问题解决:基于数据与事实、分层级协调原则、秉持“事要解决”的态度
    - 强化沟通会商:建立会商制度、倡导换位思考与主动沟通、善用非正式沟通
  - 教育培训(观/技/业/素/形)
    - 思想观念:职业道德、服务精神、树立先进典型
    - 技能学习:政策理论、专业知识、前沿动态
    - 业务演练:应急演练、技能练兵、业务研讨、案例分析
    - 综合素质:舆情应对、沟通技巧、心理疏导
    - 创新形式:线上线下结合、参观学习、组建小组


























# 应急处理
- 基本信息:偶考、350字、35分
- 特点：有场景性、作答分轻重缓急
- 解题方法:
  - 审题
    - 看是否是应急处理题
    - 看身份、目标、要求、情景、问数
  - 析点
    - 找矛盾点（问题/影响/原因）
    - 找对策（直接/间接）
  - 加工
    - 有矛盾和对策时
      - 轻重缓急排序书写
      - 问题与对策对应书写
    - 无矛盾和对策时
      - 按照套路确定小标题
      - 从文章找点填充
  - 书写
    - 按照套路，总结经验教训/向上级汇报
- 常见方案(稳/查/理/结)
  - 稳定局面
    - 接待:明确身份、立即接待
    - 隔离:分化隔离
    - 安抚:规范用语、语言安抚、情绪稳定、听取诉求、详细记录
  - 调查核实
    - 方式:成立工作组、明确分工、全面、客观、公正
    - 目的:调查研究、问题核实、实地调查、还原事情真相
  - 妥善处理
    - 对群众
      - 我方过错:不足、道歉、方案、补偿、谅解
      - 存在误会:耐心解释、寻求共识、表达诉求、赢得理解
      - 正常规定:明确告知、解释规定、积极引导、依法依规
    - 对内部
      - 问题:肯定成绩、指出错误、促成道歉、争取谅解
      - 处罚:责任人问责、员工研讨
      - 培训:业务培训、沟通技巧、心理疏导
    - 对组织
      - 整改:优化流程、严格整改（具体做法）、严肃问责
      - 监督:监管机制、日常监管、信息公开、专项整治、常态化抽查
  - 总结反馈
    - 及时回应:辟谣、引导舆论、及时反馈、公开透明、结果公开、权威发声
    - 全面总结:分析根源、总结经验教训、总结汇报
    - 宣传教育:加强政策宣传、普及知识流程、开展线上宣讲、提高配合度
    - 监测:舆情监测、后续跟进、长效机制、彻底解决














# 沟通协调
- 特点：高频，350字左右，以谈话/劝说为主
- 解决方法
  - 审题
    - 明确身份、沟通对象、沟通目的
    - 找到需要解决的矛盾
  - 析点
    - 找矛盾、找对策
  - 加工(解/害/助/求)
    - 理解：问题、同情
    - 利害:肯定、政策、好处
    - 帮助:找矛盾、找对策
    - 要求:提出需求/奖罚分明/鼓励
  - 书写
    - 要点前置：核心得分词+要点扩充
    - 篇幅合理、卷面整洁



















# 公文写作

- 必考、600字左右、40分

### 核心结构

- 标题
  - 居中；过长时上下行居中对齐；除引号、书名号、破折号、分隔号外，不出现标点符号
  - 给定标题直接使用,自拟标题(关于+发文事由+文种)[中共中央关于xxx的决议]
- 发文对象
  - 顶格书写；单独占行；后加冒号例如“尊敬的xxx:”
  - 多个对象
    - 先单位后个人；先大后小；先远后近、先疏后亲
    - 同级同类用顿号,不同类型用逗号.[各省、自治区、直辖市，国务院各有关部门，各直属单位]
- 正文
  - 开头:（左空两格）原因、背景
  - 主体:（左空两格）现状、问题、对策(与主题相关的内容都要写上)
  - 结语:（左空两格）总结、呼吁
- 发文单位:右空四格
- 发文日期:右空四格[XXXX 年 XX 月 XX 日]总共7个字

### 报告类

总结、报告、汇报、简报

```
关于 XXXX 的报告
开头：背景介绍+引出主题
主体：
  （材料为王）层层递进式，现状、问题、分析、对策
  分小标题（抓重点，用词准确简明）
结语：自然收尾/表态
发文单位(酌情)
发文日期(酌情)
```

### 宣传类

宣传稿、倡议书

```
关于……的倡议书//作文式标题（如保护环境人人有责）
发文对象（泛称）：
开头：背景、意义、目的、引出主题；
主体：宣传倡议的内容（多为做法）；
结语：呼吁号召；
发文单位
发文日期
```

### 讲话类

讲话稿、发言稿、讲解稿、经验介绍稿；致辞、演讲稿；

```
关于/在……的讲话//自拟标题（根据题目内容定）
发文对象：记得加：号
开头：问候、介绍、背景、主题；
主体：（材料为王）观点、建议、经验
结语：呼吁号召、表达谦逊、谢谢大家；
```

### 活动策划

活动、会议、策划、筹备

```
主办单位//区域+活动名称+方案
开头：背景
主体：主体、内容、方式、效果
结尾：
发文单位
发文日期
```
- 背景、主体(单位、对象、师资)、内容(时间、地点)、方式、效果（考核/评优）

### 问卷调查

导语、基本信息、核心问题

```
关于……的调查问卷
导语：背景(目的、意义)、主体(发起方、受调者)、指南(耗时/必填)、承诺(信息安全)
具体问题：
一、基本信息（姓名、年龄、性别、学历、职业、联系方式……）
二、核心问题（尽量分类；数量适中，15—20 个；逻辑层次：由浅入深、由表及里;开放性问题）
结语：致谢。
```

### 通知类

上级向下级表彰、批评、情况说明、生活中问题//事件//事迹广泛告知、政策法规、规定、公开信

```
标题：发文机关+关于+发文事由(文件内容/制度名)+通知
发文对象：按照规范排列
正文
		背景:目的、意义
		主体:参与方
		内容:时间、地点、流程、细则
```

# 分析判断

- 依据材料案例，谈谈你对材料中的某一句话或是某一观点的理解和评价；
- 频次不高、150字、15分
- 审题
  - 题干中出现“解释、理解，内涵、含义，认识,看法、见解、观点，评价、评论、评述,启示、借鉴意义”；
- 析点
  - 解释意思（转化为直白表述、不要展开，简明扼要）
  - 论证分析:背景、现状、原因、影响、本质、趋势
  - 针对做法/政策：合理性、有效性、必要性
- 加工
  - 根据题目要求分类
  - 根据文章结构暗示分类
  - 根据范围分类
- 书写
  - 分段书写
