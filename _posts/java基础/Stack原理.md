# 简介

- Stack是栈。它的特性是：**先进后出**(FILO, First In Last Out)。<!--more-->
- 继承于Vector，意味着**Stack也是通过数组实现的**



</br>

# 基本用法

- 执行**push**时(即，**将元素推入栈中**)
- 执行**peek**时(即，**取出栈顶元素，不执行删除**)
- 执行**pop**时(即，**取出栈顶元素，并将该元素从栈中删除**)
